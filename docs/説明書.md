# Crypto Relative Strength Tracker 説明書

このドキュメントは、`my-crypto-watch` の利用方法をまとめたユーザー向け説明書です。

## 1. アプリ概要
推しコインが主要暗号資産に対してどれだけ強い／弱いかを、複数の時間軸で可視化するWebアプリです。CoinMarketCap APIの最新データを利用します。

## 2. 起動方法
1. 依存関係のインストール
   - 初回のみ: `npm install`
2. 環境変数の設定
   - `/Users/daichitsuchiya/Documents/apps/ChiapuruSite/my-crypto-watch/.env.local`
   - `CMC_API_KEY` を実際のAPIキーに置き換えてください。
   - `EXCHANGE_RATE_API_KEY`（ExchangeRate-API）を追加してください。
3. 開発サーバー起動
   - `npm run dev`
4. ブラウザでアクセス
   - `http://localhost:3001` もしくは `http://127.0.0.1:3001`

## 3. 画面の見方

### 推しコイン選択
- 画面上部の「推しコイン」入力欄で検索し、任意のコインを選択します。

### 比較対象グループ
- `Top 10 / レイヤー1 / DeFi / ミームコイン / カスタム` から選択します。
- `主要法定通貨` では USD/EUR/JPY/GBP/AUD/CAD/CHF/CNY を比較できます。
- カスタムでは最大15銘柄まで自由に選べます。

### 総合相対強度ゲージ
- 推しコインが比較対象グループより強い/弱いを総合スコアで表示します。
- `-50% 〜 +50%` の範囲で表示されます。

### 相対強度バーチャート
- 0%が基準ラインです。
- 右（プラス）: 推しコインが強い
- 左（マイナス）: 推しコインが弱い
- バー右端に「推しが強い/弱い」文言が表示されます。

### 詳細比較テーブル
- 各銘柄の価格と時間軸別の変動率を表示します。
- 相対強度列は「推しコインの変動率 − 対象銘柄の変動率」です。
 - 法定通貨は `FX` と表示されます。

### 更新
- 画面右下の「更新」ボタンで再取得します。
- デフォルトは5分キャッシュです。

## 4. 注意事項
- 本アプリは投資助言を目的としたものではありません。
- CoinMarketCapの無料プランはレート制限があります。
- APIキーは必ずサーバーサイドで管理され、クライアントに露出しません。
 - ExchangeRate-APIの無料プランは日次更新のため、短期（1h/24h）の値は参考値になります。

## 5. トラブルシューティング
- 画面が開かない: `npm run dev` が起動しているか確認してください。
- ERR_CONNECTION_REFUSED: `http://127.0.0.1:3001` を試してください。
- データ取得エラー: `CMC_API_KEY` が正しいか確認してください。
